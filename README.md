# flashcards-react-native-app

Flashcards is a React Native app for iOS and Android that allows users to create decks, add cards and quiz themselves.

# Installation

```bach
npm install
```
# Usage

The project can be run with npm or yarn

```
npm start
```

or

```
npm run android
or
npm run a
```
